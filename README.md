# graffiti
A <b>canvas-based HTML5</b> drawing tool with a feature of saving the picture you drew as an <b>SVG</b>
The app works awesome with retina displays and supports shortcuts 
<i>(Ctrl + z, Ctrl + n, Ctrl + s, Ctrl + 0 :)</i>
